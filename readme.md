# rnm [![Build Status](https://travis-ci.org/alonalon/rnm.svg?branch=master)](https://travis-ci.org/alonalon/rnm)

> Change paths or filenames.


## Install

```
$ npm install -g rnm
```

## Usage

```js
rnm file.js file.jsx
```

## CLI

```
$ npm install --global rnm
```

```
$ rnm --help

  Usage
    rnm <oldpath> <newpath>

  Options
    --wildcard

  Examples
    $ rnm *.jsx .js --wildcard
    $ rnm file.jsx file.js
```
